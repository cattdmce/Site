Events

The events for Technitude 2016 are a mixture of fun and technology. These events will surely be thrilling and can give you an adrenaline rush. The list contains 9 events.

Tech Go

Tech Hunt consists of a series of clues using which the contestants will have to embark on the hunt for the ultimate treasure. The clues will be purely technical and may contain programming hints to be decoded. Each team will have 3 members.

Robo Champ and Exhibition

A tournament where each participant will be given a locomotive robot. The goal is to capture the maximum number of balls from the arena and to put them in their respective baskets. The participant with maximum balls in the basket at the end of the round will be declared as winner for that round.

Code_Wars();

Code_Wars(); is a coding contest wherein participants will have to code in C language. The objectives of the program are those from within the syllabus. This is an individual event.

Buzz It!

Buzz It! is a Quiz Competition where the first team to ring the buzzer gets the chance to answer the question. The correct answers will be rewarded with positive points. The team with maximum points will be declared as round winner.

Blind Coding

Blind Coding is an event where the participants will have to write a program code in C. The concept being that the monitor will be turned OFF with only the C compiler running. This is an individual event.

Debugging

Debugging is a coding event where the participants will be provided with the buggy code and the goal will be to eliminate all the errors in the minimum possible time.

Time - Out

Time - Out is a fun kind of event where each participant will have to complete the given tasks within a minute. There will be a set of 5 tasks with time limit of 1 minute per event.

Shut Down

Shut Down will be a coding event carried on Linux Platform. If the code contains atleast one error, then the PC will automatically Shut Down.

Night Out

In this event, the participants will be assigned the coding objective via eMail, exactly at midnight. And they will have to come up with error free running code in any language on the 2nd day of the event.
